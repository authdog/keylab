How to install easyjwt?

Installing on nodejs

easyjwt is distributed as a library available publicly on npm (node package manager).
Link to npm website

note
nodeJS 12+ is recommended to use `@authdog/easyjwt` in good conditions. You can download the version adapted to your system on nodejs website.
nodeJS releases

easyjwt can be installed via a package manager such as `yarn` or `npm`.

yarn#

yarn add @authdog/easyjwt

npm#

npm install @authdog/easyjwt
