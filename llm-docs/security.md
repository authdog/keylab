# Security

This site mirrors the repository security guidance in a docs-friendly format.

## Supported versions

Security fixes are provided for the latest published release line of `keylab`.

| Version | Supported |
| ------- | --------- |
| latest | yes |
| older | no |

## Reporting a vulnerability

Do not open a public issue or pull request for a suspected vulnerability.

Report it privately through:

- `opensource@authdog.com`
- <https://authdog.com/whitehat>

Include:

- a clear description of the issue
- affected versions
- reproduction steps or a proof of concept
- impact details, if known

## Disclosure expectations

Please allow reasonable time for investigation and remediation before public disclosure. When a fix is available, maintainers may publish:

- a patched release
- release notes
- a public advisory when appropriate
